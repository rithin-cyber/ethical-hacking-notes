# Browser Observation

Cookie:A cookie is a small piece of data stored in the user's browser by a website to remember information between HTTP requests.

Session:A session is server-side data that stores information about a user's interaction with a website, such as their login state and identity.

Headers:A Session ID is a unique identifier that links a user's browser to their session stored on the server.
