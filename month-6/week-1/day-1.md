# What I learned
^^What is HTTP?
HTTP is the application protocol that browsers and servers use to communicate and exchange web data.
^^What is an HTTP Request?
An HTTP request is a message sent from the browser to the server asking for a resource or action.
^^What is an HTTP Response?
An HTTP response is a message sent from the server back to the browser containing the requested data or result.
^^What is a Header?
Headers are additional pieces of information sent with an HTTP request or response.
^^What does TCP do?
TCP provides reliable communication between the browser and the server.
^^What does Burp Suite do?
Burp Suite sits between the browser and the server, allowing you to intercept, inspect, modify, and forward HTTP requests and responses.
